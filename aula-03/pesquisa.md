## 1. O que é uma estrutura condicional em programação? Dê um exemplo do cotidiano usando "se... então... senão".
É uma estrutura que permite tomar decisões no programa. Exemplo: se estiver chovendo, então levo guarda-chuva; senão, saio sem guarda-chuva.

## 2. Qual é a diferença entre usar apenas if e usar if/else?
if executa um bloco apenas quando a condição é verdadeira. if/else executa um bloco quando a condição é verdadeira e outro quando é falsa.

## 3. Quando usamos else if? Crie um exemplo com três ou mais possibilidades.
Usamos else if quando existem várias possibilidades. Exemplo: nota >= 9 = Excelente; nota >= 7 = Bom; nota >= 6 = Regular; senão = Reprovado.

## 4. O que significa uma condição retornar true ou false?
Significa que o resultado da condição é verdadeiro (true) ou falso (false).

## 5. Qual é a diferença entre == e === em JavaScript? Qual deles é mais recomendado?
== compara apenas os valores. === compara valor e tipo. O mais recomendado é ===.

## 6. Para que servem os operadores &&, || e !? Explique com um exemplo para cada um.
&& significa E; || significa OU; ! significa NÃO.

## 7. O que o operador % faz? Por que ele é útil para descobrir se um número é par ou ímpar?
O operador % retorna o resto da divisão. Ele ajuda a descobrir se um número é par ou ímpar.

## 8. O que é switch case? Em que situação ele fica melhor do que vários else if?
switch case é uma estrutura usada para escolher entre várias opções. É melhor que vários else if quando comparamos a mesma variável com diferentes valores.

## 9. Para que servem break e default dentro de um switch?
break encerra o caso atual. default é executado quando nenhum caso corresponde ao valor informado.

## 10. Por que é importante testar o mesmo programa com vários valores diferentes?
Para verificar se o programa funciona corretamente em diferentes situações e encontrar possíveis erros.
