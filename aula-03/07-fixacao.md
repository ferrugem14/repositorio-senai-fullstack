## 1. Qual é a diferença entre condição e decisão?

Condição é o teste que o programa faz para verificar se algo é verdadeiro ou falso. Decisão é a ação que o programa toma com base no resultado da condição.

## 2. Por que a ordem dos else if pode mudar o resultado?

Porque o programa verifica as condições de cima para baixo e para na primeira que for verdadeira.

## 3. Quando é melhor usar switch em vez de else if?

Quando precisamos comparar a mesma variável com vários valores diferentes, deixando o código mais organizado.

## 4. O que acontece se esquecer o break dentro do switch?

O programa continua executando os próximos casos do switch, mesmo que eles não correspondam ao valor da variável.

## 5. Qual foi o erro mais comum que a dupla encontrou durante a prática? Como corrigiu?

O erro mais comum foi declarar a mesma variável mais de uma vez. Corrigimos removendo a declaração repetida.
