# Pesquisa - Estruturas de Repetição

## 1. O que é uma estrutura de repetição?
É uma estrutura usada na programação para executar um bloco de código várias vezes, enquanto uma condição for verdadeira ou por uma quantidade definida de vezes.

## 2. Para que serve o laço for?
Serve para repetir um bloco de código quando se sabe previamente quantas vezes a repetição deve acontecer.

## 3. Para que serve o laço while?
Serve para repetir um bloco de código enquanto uma condição for verdadeira.

## 4. Para que serve o laço do-while?
Serve para repetir um bloco de código enquanto uma condição for verdadeira, mas garante que o código seja executado pelo menos uma vez.

## 5. Qual é a diferença entre for, while e do-while?
while é usado quando a repetição depende de uma condição. Já o do-while é semelhante ao while, mas executa o bloco de código pelo menos uma vez antes de verificar a condição.
