# Pesquisa - Operadores JavaScript

## 1. O que são operadores em programação?
Operadores são símbolos utilizados para realizar operações com valores e variáveis dentro de um programa.

## 2. Para que servem os operadores aritméticos? Cite exemplos.
Servem para realizar cálculos matemáticos.

Exemplos:
- + (adição)
- - (subtração)
- * (multiplicação)
- / (divisão)
- % (resto da divisão)

## 3. Qual a diferença entre / e % em JavaScript?
O operador / realiza uma divisão e o operador % retorna o resto da divisão.

## 4. Como descobrir se um número é par usando o operador %?
Um número é par quando o resto da divisão dele por 2 é igual a 0.

## 5. O que é uma expressão em programação?
É uma combinação de valores, variáveis e operadores que gera um resultado.

## 6. O que significa precedência de operadores?
É a ordem em que os operadores são executados em uma expressão.

## 7. Por que os parênteses podem mudar o resultado de uma expressão?
Porque as operações dentro dos parênteses são executadas primeiro.

## 8. Qual a diferença entre =, == e ===?
= é usado para atribuição de valor.
== compara apenas os valores.
=== compara os valores e os tipos dos dados.

## 9. Por que é recomendado usar === em JavaScript?
Porque ele faz uma comparação mais segura, verificando valor e tipo ao mesmo tempo.

## 10. O que os operadores relacionais retornam: texto, número ou boolean?
Retornam valores booleanos: true ou false.

## 11. Quando usamos o operador >? E o operador >=?
O operador > é usado para verificar se um valor é maior que outro.
O operador >= é usado para verificar se um valor é maior ou igual a outro.

## 12. Quando usamos o operador <? E o operador <=?
O operador < é usado para verificar se um valor é menor que outro.
O operador <= é usado para verificar se um valor é menor ou igual a outro.

## 13. Para que serve o operador lógico &&?
Serve para verificar se duas condições são verdadeiras ao mesmo tempo.

## 14. Para que serve o operador lógico ||?
Serve para verificar se pelo menos uma condição é verdadeira.

## 15. Para que serve o operador lógico !?
Serve para inverter um valor lógico.

## 16. Qual a diferença entre uma condição simples e uma condição composta?
Uma condição simples possui apenas uma comparação. Uma condição composta possui duas ou mais comparações ligadas por operadores lógicos.

## 17. Explique uma situação real em que duas condições precisam ser verdadeiras ao mesmo tempo.
Para entrar em uma festa, a pessoa precisa ter ingresso e apresentar um documento de identificação.

## 18. Explique uma situação real em que apenas uma entre duas condições precisa ser verdadeira.
Uma compra pode ser paga com cartão ou PIX.

## 19. Como o console.log() pode ajudar a encontrar erros no código?
Ele permite visualizar valores e resultados durante a execução do programa, ajudando a identificar problemas.

## 20. Por que devemos testar o mesmo código com diferentes valores?
Porque isso ajuda a verificar se o programa funciona corretamente em diferentes situações e evita erros inesperados.
